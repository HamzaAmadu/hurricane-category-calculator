# Hurricane Category Calculator

A Python application that classifies storm intensity based on wind speed according to the Saffir-Simpson Hurricane Wind Scale.

#Features
- Classifies hurricanes from Category 1 to 5
- Identifies tropical storms and non-storm conditions
- Robust error handling for invalid inputs
